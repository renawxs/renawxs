**`Estudante de Análise e Desenvolvilmento de Sistemas`**

Olá! 👋 Sou o Renan Santos, tenho 20 anos nasci e moro no Rio de Janeiro.

Minha história com a tecnologia começou cedo, desde criança os videogames e computadores nunca foram apenas diversão, mas um quebra-cabeça que eu queria entender como funcionava. Quando chegou a hora de escolher uma carreira, a computação não foi uma dúvida, foi o meu primeiro e único caminho.

Atualmente, sou estudante de Análise e Desenvolvimento de Sistemas (ADS) e já atuo na área com foco em crescimento contínuo. Minha principal paixão é criar experiências visuais incríveis e intuitivas como Web Designer / Front-end, mas sei que o verdadeiro impacto acontece quando a interface se conecta a uma arquitetura robusta. Por isso, também construo bases sólidas no Back-end, buscando sempre uma visão completa (Full Stack) de cada projeto.
